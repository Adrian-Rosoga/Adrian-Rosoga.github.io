Site

The webpage automates the task of answering "How's the weather in London?". Our parents live abroad and very often were asking this. Now they can simply look outside via 2 Raspberry Pi cameras installed at the windows.

Under the hood there is a fair amount of tech

  • ngrok - To escape from the Hyperoptic CGNAT
  • Netlify - Hosts this web site
  • Raspberry Pi - Cameras and Gunicorn/Flask web servers
  • AWS Elastic Beanstalk - Thames Tide Graph
  • AWS Lambda and AWS API Gateway - Thames Tide Graph
